About HematoVision

HematoVision is a deep learning-based blood cell classification system. It helps in identifying cell types such as RBC, WBC, Platelets, and abnormal cells.

Objectives

  • Automate blood cell classification
  • Use Transfer Learning for higher accuracy
  • Support healthcare diagnostics
  • Reduce manual microscopic work

Tools & Technologies

  • Python
  • TensorFlow / Keras
  • Transfer Learning Models
  • Flask Web Framework
  • HTML + CSS
